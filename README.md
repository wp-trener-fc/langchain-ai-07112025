### przygotowanie srodowiska wirtualnego

1. przy uzyciu pythona 3.12.8 utworz zrodowisko wirtualne w fodlerze projketu
```bash
path\to\python3128\python.exe -m venv nazwa_srodowiska
```

2. przy uzyciu pythona 3.12.8 utworz zrodowisko wirtualne
```bash
nazwa_srodowiska\Scripts\activate
```

3. zainstaluj biblioteki z pliku requirementx.txt
```bash
pip install -r requirements.txt
```
